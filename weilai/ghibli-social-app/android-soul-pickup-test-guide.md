操作：手指按住卡片不动

### 🎮 交互测试流程

1. 选择任意卡片
2. 长按800ms直到触发拾取
3. 观察完整动画序列
4. 释放手指完成拾取

#### 测试场景 B：提前释放
1. 长按500ms (不足800ms)
2. 提前释放手指
3. 验证动画回退到初始状态
4. 确认无拾取动作触发

#### 测试场景 C：滑动取消
1. 开始长按
2. 手指滑动离开卡片区域
3. 验证动画取消
4. 确认拾取动作未触发

### 🐛 常见问题及解决方案

#### 问题1：动画不触发
```
症状：长按无反应
排查：
1. 检查触摸事件是否正常传递
2. 验证长按时间设置 (GestureHandler)
3. 确认动画状态机逻辑
4. 检查Reanimated工作线程状态
```

#### 问题2：粒子效果异常
```
症状：粒子不显示或显示异常
排查：
1. 检查粒子组件渲染逻辑
2. 验证动画数值范围
3. 确认样式属性正确性
4. 测试不同屏幕密度适配
```

#### 问题3：色彩显示错误
```
症状：颜色不匹配或显示异常
排查：
1. 验证颜色值格式 (RGB vs HSL)
2. 检查渐变组件配置
3. 确认透明度计算正确
4. 测试色彩空间转换
```
