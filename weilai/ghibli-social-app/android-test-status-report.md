```bash

cd android && ./gradlew clean
cd ..
npm install
npx expo prebuild --clean
npx expo run:android
```

```bash

```

### 常见问题3: 动画不流畅
```
可能原因:
- 设备性能不足
- 后台应用过多
- 省电模式开启
- 调试模式影响

解决方案:
- 关闭后台应用
- 关闭省电模式
- 使用真机测试
- 清理设备缓存
```
