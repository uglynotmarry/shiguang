4. 观察以下动画效果：

## 🐛 常见问题排查

### Metro bundler 问题
```bash
# 如果端口被占用
npm start -- --port 8082

# 清除缓存
npx expo start --clear

# 重置Metro缓存
npx react-native start --reset-cache
```

### Android 构建问题
```bash
# 清理构建缓存
cd android && ./gradlew clean

# 重新安装依赖
npm install
npx expo prebuild --clean
```

### 应用崩溃排查
1. 检查 Android Studio 日志
2. 查看 Metro bundler 错误输出
3. 验证所有依赖包版本
4. 检查设备兼容性
