# Android Studio 快速测试指南

## 🚀 快速开始

### 1. 打开Android Studio
1. 启动Android Studio
2. 选择"Open an Existing Project"
3. 导航到：`C:\Users\ugly\Desktop\shiguang\ghibli-social-app`
4. 等待项目加载完成

### 2. 连接设备

#### 选项A：使用Android模拟器
1. 点击Android Studio顶部的"Device Manager"
2. 点击"Create Device"
3. 选择设备（推荐：Pixel 4）
4. 选择系统镜像（推荐：Android 11或更高）
5. 点击"Finish"并启动模拟器

#### 选项B：使用真实设备
1. 启用手机的"开发者选项"
2. 启用"USB调试"
3. 用USB线连接电脑
4. 在手机上授权USB调试

### 3. 构建和运行

#### 方法一：使用Android Studio（推荐）
1. 等待Gradle同步完成（底部进度条）
2. 点击绿色"Run"按钮（或按Shift+F10）
3. 选择目标设备
4. 等待应用安装和启动

#### 方法二：使用命令行
```bash
# 在项目目录下执行
cd C:\Users\ugly\Desktop\shiguang\ghibli-social-app
npx react-native run-android
```

### 4. 测试3D球体界面

#### 基本功能测试
1. 应用启动后，你会看到"🌐 灵魂星球"界面
2. 球体上分布着6个彩色节点
3. 每个节点显示不同的情感内容

#### 交互测试
- **触摸测试**：点击不同的节点
- **视觉反馈**：观察节点的颜色和位置
- **界面响应**：检查界面流畅度

## 🔧 故障排除

### 设备未识别
```bash
# 检查设备连接
adb devices

# 重启ADB服务
adb kill-server
adb start-server
```

### 构建失败
```bash
# 清理构建缓存
cd android
./gradlew clean
cd ..
npx react-native run-android
```

### 应用崩溃
1. 查看Android Studio的Logcat
2. 检查错误日志
3. 确保所有依赖正确安装
