# Android Studio 灵魂拾取动画演示项目

## 快速开始指南

### 选项1：WebView演示（5分钟设置）

1. **在Android Studio中创建新项目**：
   - File → New → New Project
   - 选择 "Empty Activity"
   - Name: "GhibliSoulDemo"
   - Package: "com.example.ghibisouldemo"
   - Language: Java

2. **替换生成的文件**为我提供的文件：
   - `MainActivity.java` → 使用我提供的 `AndroidWebViewDemo.java`
   - `activity_main.xml` → 使用我提供的布局文件
   - `AndroidManifest.xml` → 添加网络权限

3. **运行项目** - 会自动加载灵魂拾取测试页面
